Import Custom Distribution

WSL te permite instalar cualquier sistema Linux si tienes sus archivos en un archivo comprimido de tipo .tar o .tar.gz (llamado rootfs).

Lo que necesitas para importar:

  • Nombre: Un nombre corto para tu Linux (ej. MiAlpine).
  • Carpeta de instalación: Dónde se guardarán los archivos (ej. C:\WSL\MiAlpine).
  • Archivo Tarball: La ruta del archivo comprimido .tar en tu PC.

¿Cómo conseguir un archivo .tar?

  1. Copia una de tus distros: Abre PowerShell y corre (sustituye Ubuntu por el nombre de tu distro):
    wsl --export Ubuntu C:\WSL\mi-copia.tar (ejemplo)
  2. Exporta desde Docker: Si usas Docker, puedes exportar cualquier contenedor a tar (sustituye contenedor por el nombre real):
    docker export contenedor -o C:\WSL\contenedor.tar (ejemplo)
  3. Descargas oficiales: Webs como la de Alpine Linux ofrecen su Mini Root Filesystem directo en .tar.gz para descargar. No necesitas escribir ningún comando.
